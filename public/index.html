API SobraZero

Descripción

La API de SobraZero permite gestionar la información de comercios y ofertas de alimentos con descuentos. Su objetivo es reducir el desperdicio de comida conectando panaderías, supermercados, verdulerías, rotiserías y restaurantes con usuarios que buscan ahorrar entre un 50% y un 70% en productos en perfecto estado, cercanos a su fecha de vencimiento.

A través de sus endpoints principales (/comercios y /ofertas), la API facilita:

  • Visualizar todos los documentos de una colección.
  • Obtener un documento específico por ID.
  • Crear nuevas ofertas o comercios.
  • Actualizar la información de un documento existente.
  • Eliminar registros.

De esta manera, SobraZero API brinda una base técnica que respalda el funcionamiento de la aplicación, asegurando transparencia, ahorro real y un impacto positivo en la reducción del desperdicio de alimentos.

Endpoints:

Endpoints

  • GET /api/usuarios
  • GET /api/comercios
  • GET /api/ofertas
